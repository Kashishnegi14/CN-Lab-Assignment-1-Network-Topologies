How to Test

Open Cisco Packet Tracer.

Load the exp1_topologies.pkt file.

Switch to Simulation Mode to observe packet flow.

Perform ping tests between PCs to verify connectivity.

Disconnect one switch link in the ring topology to test fault tolerance.

Description

This experiment demonstrates the implementation of Star, Hub-based, and Ring-like network topologies.

It compares their connectivity, performance, and fault tolerance by observing packet transmission behavior and network response under normal and failure conditions.
