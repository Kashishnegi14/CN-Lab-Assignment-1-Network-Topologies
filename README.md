How to Test
Open Cisco Packet Tracer.
Load exp1_topologies.pkt file.
Use Simulation Mode to observe packet flow.
Perform ping tests between PCs.
Disconnect one switch link in ring topology to test fault tolerance.
Description
This experiment demonstrates Star, Hub-based and Ring-like topologies and compares their connectivity, performance and fault tolerance.
